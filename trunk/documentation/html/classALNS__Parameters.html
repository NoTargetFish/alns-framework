Member Enumeration Documentation

enum ALNS_Parameters::StoppingCriteria

Enumeration representing the various kind of stopping criteria. MAX_IT: the maximum number of iterations. MAX_RT: the maximum run time. MAX_IT_NO_IMP: the maximum number of iterations without improvement. ALL: a mix of the MAX_IT, MAX_RT and MAX_IT_NO_IMP.
